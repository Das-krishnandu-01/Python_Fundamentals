# Python_Playground
This repository contains a collection of Python scripts from my Data Analysis and Programming lab course, covering Units 1 through 5.

## Program Topics Covered

### Unit 1: Python Basics
* **P_1_1_kg_to_lbs.py**: Converts weight from kilograms to pounds.
* **P_1_2_for_loop_sequence.py**: Prints a number sequence using a `for` loop.
* **P_1_3_string_to_list.py**: Splits a string into a list of characters.
* **P_1_4_list_max.py**: Finds the largest number in a list.
* **P_1_5_fibonacci.py**: Calculates the nth Fibonacci number recursively.

### Unit 2: Object-Oriented Programming (OOP)
* **P_2_1_car_class.py**: A simple `Car` class with attributes and methods.
* **P_2_2_inheritance.py**: Demonstrates inheritance with `Animal` and `Cat` classes.
* **P_2_3_polymorphism.py**: Shows polymorphism with different `Animal` subclasses.
* **P_2_4_error_handling.py**: Uses `try-except` blocks for error handling.

### Unit 3: NumPy
* **P_3_1_numpy_basics.py**: Basic NumPy array creation and analysis.
* **P_3_2_numpy_analysis.py**: Advanced array filtering and finding min/max values.

### Unit 4: Pandas
* **P_4_1_pandas_series.py**: Creating and converting Pandas Series.
* **P_4_2_pandas_dataframe.py**: Creating, modifying, and inspecting Pandas DataFrames.

### Unit 5: Matplotlib
* **P_5_1_series_of_plots.py**: Creates a figure with multiple line plots.
* **P_5_2_subplot_layout.py**: Generates a subplot grid with scatter, line, and bar plots.
* **P_5_3_time_series_visualization.py**: Visualizes time-series data with custom labels.
* **P_5_4_3d_plot.py**: Creates a 3D scatter plot.

## How to Use
Each `.py` file is a standalone script. You can run any file using a Python interpreter, for example:
`python P_1_1_kg_to_lbs.py`
